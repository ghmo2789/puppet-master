The Project

Learn offensive security to become better at defensive

The main goal of this project is to build a tool that allows you to control a network of computers, build an interface used to control the network and generally increase security awareness among developers. The tool is a server management system that consists of a web GUI with an associated web server, a control server software, and a client application, that together will allow a single user to control many computers from the web GUI. The system could, for example, be used for distributed computations, extracting or sending files to the controlled clients, or generating web traffic from multiple hosts to one website of choice. In other words, the tool that is going to be built during this project can be seen as a botnet manager that is intended to be used for educational and research purposes. The goal is to build an easy-to-use tool that will allow developers or system administrators to test their own systems and make them a bit safer against the cyber security threats that come from botnets.
